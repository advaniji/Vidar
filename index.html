Installation & Usage Guide

Follow these steps to get started with Vidar Downloader:

  1. Download the Files: Get vidar.py, yt-dlp.exe, and the optional aria2c zip file.
  2. Extract aria2c: If you downloaded aria2c, extract aria2c.exe and place it in the same directory as the other files.
  3. Prepare the Environment: Ensure all files reside in one folder.
  4. Install Dependencies: Run: 
    pip install customtkinter
  5. Run Vidar Downloader: Execute vidar.py using Python 3.
  6. Using the Application: Enter the video URL (and password if needed) in the GUI. For advanced performance, enable Turbo Mode in the settings panel.

Ensure you have Python 3 installed. For additional details or troubleshooting, please consult the project documentation on GitHub.
