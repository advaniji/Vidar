Installation & Setup

To get started with Vidar Downloader, please follow these steps:

  1. Download the vidar.py file along with the dependencies: yt-dlp.exe and the aria2c zip file.
  2. Extract the aria2c.exe from the downloaded zip file and place it in the same directory as vidar.py and yt-dlp.exe.
  3. Ensure all files are in the same folder for proper functionality.
  4. Install the required Python dependency: 
    pip install customtkinter
  5. Run vidar.py using Python 3.

Make sure you have Python 3 installed on your system. If you encounter any issues, please refer to the repository documentation.

About & Credits

Vidar Downloader is a minimal yet powerful video downloader built with Python. It leverages yt-dlp for efficient video downloads and uses a modern, intuitive GUI built with CustomTkinter. The optional integration with aria2c further accelerates downloads.
